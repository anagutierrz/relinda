# Relinda Web v24 — Secciones comerciales

Base: diseño aprobado v23.

Incluye:
- Relinda Club / suscripciones.
- Selector de compra única o suscripción en páginas de producto.
- Página Conviértete en Distribuidor Oficial Relinda.
- Página Conviértete en Embajador(a) Oficial Relinda.
- Home actualizada con previews de Club, Distribuidores y Embajadores.
- Footer y navegación actualizados.

Notas para WordPress + WooCommerce:
- Relinda Club debe implementarse con productos de suscripción / frecuencia de envío.
- Compra única vs Suscribirme y ahorrar debe conectarse al selector real de WooCommerce.
- Formularios deben conectarse a Fluent Forms, Gravity Forms, HubSpot, Zoho CRM o el CRM elegido.
- Área privada de distribuidores debe implementarse con roles de usuario, contenidos restringidos y dashboard de pedidos/facturas.
